# 使用指南

## 1. 安装与启动

1. 安装依赖：

```bash
pip install -r requirements.txt
```

2. 启动应用：

```bash
python main.py
```

也可以直接双击 `scripts/启动.bat`。

## 2. 首次配置

应用首次运行后，请在“媒体库设置”中添加电影目录，或手动编辑 `data/config.json`：

```json
{
  "movie_paths": [
    "Z:\\Movies"
  ]
}
```

## 3. 本地文件结构建议

每部电影建议至少包含：

- 视频文件（如 `.mkv`/`.mp4`）
- NFO 文件（与视频同目录）
- `poster.jpg`（海报）
- `fanart.jpg`（可选）

## 4. 常用功能

- 海报墙浏览与筛选
- 电影详情展示
- 一键调用系统播放器
- 收藏与观看状态管理
- 媒体库重扫

## 5. 数据文件位置

运行数据统一保存在 `data` 目录：

- `data/config.json`
- `data/movie_cache.json`
- `data/watch_history.json`
- `data/favorites.json`
- `data/failed_movies.json`
- `data/scrape_config.json`
- `data/douban_cookies.pkl`

## 6. 常见问题

1. 启动后没有电影
   - 检查 `data/config.json` 中 `movie_paths` 是否正确。
   - 在设置中点击“重新扫描媒体库”。

2. 海报不显示
   - 检查影片目录下是否存在 `poster.jpg`。

3. 播放无反应
   - 检查系统默认播放器是否可用。
