# 自定义标签功能说明

## 1. 功能简介

支持从 NFO 中读取自定义标签并用于筛选展示。

## 2. 数据来源

标签来源于 NFO 中的相关字段，解析后写入电影模型。

## 3. 界面行为

1. 自动汇总标签集合
2. 在筛选区显示标签项
3. 点击标签后仅展示匹配电影

## 4. 使用建议

1. 保持标签命名一致（避免同义词碎片化）
2. 中英文标签尽量分层使用
3. 定期清理不再使用的标签

## 5. 常见问题

1. 标签未显示
   - 检查 NFO 中是否存在对应字段
   - 重扫媒体库后再查看

2. 标签筛选结果异常
   - 检查标签大小写和空格差异
