# 打包说明

## 1. 推荐方式

在 Windows 下执行：

```bash
scripts\\build.bat
```

脚本会自动检查 PyInstaller 并打包 `onedir` 产物。

## 2. 打包输出

默认输出目录：

- `dist/`

可执行文件和依赖会在同一分发目录下。

## 3. 运行时数据

应用运行时数据位于 `data` 目录：

- `data/config.json`
- `data/movie_cache.json`
- `data/watch_history.json`
- `data/favorites.json`
- `data/failed_movies.json`
- `data/scrape_config.json`
- `data/douban_cookies.pkl`

## 4. 分发建议

1. 分发整个 `dist` 目录，不要只拷贝 exe。
2. 若升级版本，可迁移旧版 `data` 目录到新版。
3. 保持 `logo`、`styles` 等资源目录与程序同级。

## 5. 常见问题

1. 启动慢
   - 首次启动属于正常现象。

2. 报毒
   - 建议在可信环境签名并加入白名单。

3. 资源缺失
   - 检查打包配置是否包含 `styles`、`logo`、`data`。
