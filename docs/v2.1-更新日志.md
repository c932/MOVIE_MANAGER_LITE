# v2.1 更新日志

## 版本信息

- 版本：v2.1
- 类型：功能优化与体验改进

## 更新内容

1. 图片识别规则增强
   - 支持标准命名与带前缀命名

2. 图片加载稳定性改进
   - 修复缩放参数相关问题

3. 详情页布局优化
   - 信息层级更清晰
   - 交互反馈更明确

## 数据路径说明（当前）

数据文件统一在 `data` 目录，例如：

- `data/config.json`
- `data/movie_cache.json`
- `data/watch_history.json`
- `data/favorites.json`

## 升级建议

1. 备份 `data` 目录
2. 更新程序文件
3. 启动后执行一次重扫
