# 刮削功能说明

## 1. 功能概览

刮削功能用于补全或更新电影元数据，主流程包括：

1. 调用 tinyMediaManager 扫描与刮削
2. 豆瓣评分补全（可选）
3. 刮削完成后自动触发媒体库重扫

## 2. 入口

在主界面工具栏点击“刮削”按钮打开工作流对话框。

## 3. 工作模式

### 3.1 普通模式（requests）

- 速度较快
- 可能受反爬影响

### 3.2 Selenium 模式

- 成功率更高
- 需要安装依赖：

```bash
pip install selenium webdriver-manager
```

## 4. 登录与 Cookie

启用豆瓣登录后，Cookie 会保存到：

- `data/douban_cookies.pkl`

## 5. 配置持久化

对话框配置会保存到：

- `data/scrape_config.json`

包括抓取模式、延迟、登录选项、TMM 路径等。

## 6. 结果刷新

流程结束后会触发强制重扫，界面会更新最新评分和新增条目。

## 7. 排障建议

1. TMM 执行失败
   - 检查 TMM 安装路径和可执行权限。

2. 豆瓣匹配率低
   - 切换到 Selenium 模式。
   - 适当提高请求延迟。

3. 登录后仍失败
   - 删除 `data/douban_cookies.pkl` 后重新登录。
